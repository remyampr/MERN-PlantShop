Develop E-Commerce Site with MERN

Project Setup (2 Marks) :

  • Create a proper folder structure (client/server).
  • Configure .env for environment variables.
  • Setup Git repository.

    • Backend Development (6 Marks) :
  • Node.js and Express server setup.
  • Create RESTful APIs for products, users, and orders.
  • Connect to MongoDB using Mongoose.
    • Frontend Development (6 Marks) :
  • Create components using React.js (Home, Products, Cart, etc.).
  • Use React Router for navigation
  • Design a user-friendly interface.
    • CRUD Operations (4 Marks) :
  • Implement product management (add, edit, delete products).
  • Show product details page with dynamic routing.
    • Authentication (3 Marks) :
  • Use JWT tokens for secure authentication.
  • Implement protected routes for admin and user access.
    • Payment Integration (optional) :
  • Integrate a payment gateway (test mode).
  • Ensure payment flow is smooth and error-free.
    • Deployment (4 Marks)
  • Deploy using Vercel.
